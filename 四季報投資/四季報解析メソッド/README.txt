四季報解析メソッド

内容
- scripts/parse_shikiho_text.py: 四季報コピペテキストを構造化データへ変換するスクリプト。
- results/parsed_summary.csv: パース済みサマリ。
- results/4q2_selected_candidates.csv: 解析結果から採用した銘柄。
- results/4q2_scored_universe.csv: 解析対象全体の評価一覧。
- results/4q2_selected_to_2026_03_01.csv: 選定銘柄の 2026-03-01 までの推移。
- results/4q2_selected_to_latest.csv: 選定銘柄の最新終値ベース推移。
- criteria/採用基準まとめ.txt: 採用ロジックの説明。
